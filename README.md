# 🌙 NightSafe: Smart Campus Guardian

> Smart tracking for safer student journeys

---

## 🚨 Problem

Students often move across large campuses—especially at night—without any structured system to ensure their safety.

- No real-time tracking during walks  
- No quick way to notify guardians  
- Lack of campus-focused safety systems  
- Delayed response in emergencies  
- No record of past journeys  

This creates uncertainty for students and concern for their families.

---

## 💡 Solution

**NightSafe** introduces a simple and intelligent system to make campus movement safer and more structured.

- 📍 **Walk Tracking** – Start a session and monitor movement  
- 🧭 **Route Visualization** – Guided paths across campus  
- 🟢 **Live Status Updates** – Walking → Arrived Safely  
- 📊 **Historical Logs** – Track past journeys  
- 👨‍👩‍👦 **Extendable Alerts** – Future support for guardian notifications  

---

## ✨ Features

- 🚀 Minimal, futuristic UI  
- 📍 Embedded Google Maps integration  
- 🔄 Simulated real-time tracking  
- 📊 Instant log updates (Travel Archive)  
- ⚡ Fast and optimized performance  
- 🧠 Safe state handling (no runtime crashes)

---

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS  
- **Backend:** API Routes / Node.js  
- **Database:** Local Storage / JSON (extendable to MongoDB)  
- **Maps:** Google Maps Embed  
- **Animations:** CSS / JS-based route simulation  

---

## 🧭 How It Works

1. User enters **origin & destination**
2. Clicks **Start**
3. System:
   - Starts timer  
   - Simulates movement on map  
   - Updates status in real-time  
4. On completion:
   - Marks **Arrived Safely ✅**  
   - Stores journey in logs

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/Aash-hasFallen/nightsafe.git
cd nightsafe
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the project
```bash
npm run dev
```

### 4. Open in browser
```
http://localhost:3000
```
